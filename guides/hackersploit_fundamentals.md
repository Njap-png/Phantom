# Cybersecurity Fundamentals — HackerSploit Study Guide

Source: HackerSploit — "Top 5 Skills to Learn to Get Into Cybersecurity"
URL: https://youtube.com/watch?v=Kx4y9c7w2JQ

## Top 5 Skills

### 1. Building and Using Virtual Machines
- VMs run on top of existing "host" OS via a "hypervisor"
- Hypervisors: Hyper-V (Microsoft), VirtualBox (Oracle), VMware, KVM
- "Cloud" = thousands of VMs on specialized hypervisors in datacenters
- Key: be platform-agnostic, comfortable with ALL operating systems
- Use VMs for: training, sandboxing, analyzing suspect files, exploit testing
- Snapshots let you restore to an earlier state

### 2. Learn the Command Line
- CLI = Shell = most efficient way to interact with an OS
- Shell wraps around the Kernel (the core of the OS)
- Bash = Bourne-again Shell, default on most Linux distros
- Zsh = Z-Shell (macOS default, nicer features)
- PowerShell = go-to native shell for Windows
- WSL (Windows Subsystem for Linux) runs Linux shells natively on Windows
- CLI enables scripting, automation, and tool access without GUIs

### 3. System Administration
- Configuring and maintaining computers (personal devices to servers)
- Know your platform and tools inside and out
- Push limits: delete + recover files, analyze malware in VMs, extract passwords
- Learn by doing: read guides, follow along, practice daily

### 4. Computer Networking
- The "laws of physics" for cyberspace
- Two models: TCP/IP (4 layers) and OSI (7 layers)
  - TCP/IP: network access, internet, transport, application
  - OSI: physical, datalink, network, transport, session, presentation, application
- Layers help describe "what's happening where" — like postal service logistics
- Strong networking = rockstar troubleshooting in red, blue, or IT ops

### 5. Personal Digital Security
- Passwords, encryption, secure communications
- Stay current with security news and best practices
- Cybercrime industry is booming — IoT expands attack surface
- Practice what you preach: many breaches originate from IT/SOC staff
- Be the subject-matter expert others turn to for advice

## Key Takeaways
- Skills > Certs. What you can DO > what you KNOW
- Be versatile across platforms, not locked into one
- Automation via CLI makes you valuable
- Networking knowledge lets you "see the matrix"
- Security starts with yourself
